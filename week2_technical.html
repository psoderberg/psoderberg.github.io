CSS: Block vs. Inline-Block Styling

I've explored some differences between inline and inline-block styling. First some example, for visual learners out there. This example came from Dustin Boswell's site.

So inline-block elements act, surprisingly enough, a little like blocks and a little like inline elements. They can have width and height, but do not have a line break after them as blocks do. They are a good way to display many elements, and since they display inline, they are naturally a little responsive, so that as the browser window narrows, fewer are shown in each row.
